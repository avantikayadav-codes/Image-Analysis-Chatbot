# 🤖 Image Analysis Chatbot - Python

An AI-powered image analysis chatbot built with Python that allows users to 
upload images and ask questions about them using natural language.

---

## ✨ Features

- 🖼️ **Image Upload** — Upload any image for analysis
- 💬 **Chat Interface** — Ask questions about uploaded images
- 🤖 **AI Powered** — Intelligent image understanding
- 🔍 **Image Analysis** — Detailed image description & analysis
- 📝 **Conversation History** — Track your chat history

## ⚙️ How to Run

1. Clone this repository
```bash
git clone https://github.com/avantikayadav-codes/Image-Analysis-Chatbot.git
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run the application
```bash
python imageproject.py
```
